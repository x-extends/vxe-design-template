# Vxe 可视化设计器模板

一个基于 [Vxe UI](https://vxeui.com) 可视化设计器模板，支持任意添加自定义控件、支持任意扩展属性、轻松搭建低代码平台。
一切皆可扩展，没有什么是不能自定义，如果有就是没看文档。

## 版本

基于 Vxe v4.x 对应 vue3.x 版本
